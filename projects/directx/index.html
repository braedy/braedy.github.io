Culling the view frustum

  • Technologies: DirectX SDK, HLSL, C++, Visual Studio
  • Year: Summer 2015 (Ongoing)


This page reflects upon a personal project I am undertaking in order to better my ability programming graphics. I have also started this project to learn DirectX as it has a wider use in the videogame industry than OpenGL.

Since recent updates to the SDK, there is yet to be a wealth of good tutorial resources. Therefore much I am learning on my own, basing my implementations on previous DX11 tutorials, however utilising the new libraries (i.e. DirectXMath).

Progress thus far:

  • Direct3D device setup
  • Reading input with DirectInput
  • 3D model rendering
  • Texture mapping
  • Phong shading (diffuse, ambient, specular lighting)
  • 2D graphic rendering
  • Text rendering
  • Frustum culling
