Initial drafting of UI design

  • Technologies: Java, Swing
  • Course: GUI Programming I / Gränssnittsprogrammering I
    @ Uppsala University, Sweden
  • Role: Project Manager
  • Year: Spring 2014


The focus of the software developed within the course was the implementation aspects of user interfaces.
Within the scope of the project I learned a detailed view of different strategies and techniques related to user interface design; where the focus is on the development and implementation – rather than the graphical design.

This project involved a lot of work. I took role of Project Manager early on and it became clear that many of my team of 6 had little or no programming experience. I managed to organise and assign tasks in such a way that was supportive of them learning Java, while a lot of the initially more intricate workload was split between myself and one other team member as we had written Java Swing applications before.

Work done was related to MVC architecture, multi-threading, interrupt control, language independence, design patterns, shortcuts, actions and renderers.
